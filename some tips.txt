"""
HOSTING DJANGO WITH XAMPP LOCAL APACHE SERVER (using with Apache 2.4)
http://blog.kimlindberg.com/post/51627205148/deploying-django-with-xampp-or-apache

Download link >>>> http://www.lfd.uci.edu/~gohlke/pythonlibs/
NB: mod_wsgi can be found in the quick start pack


1.Setup a working django project
2.Install XAMPP
3.Download the specific mod_wsgi.so file that matches your python version and apache version
4.Rename it to mod_wsgi.so and move it into Apache modules folder
5.Create a django project and move it to apache htdocs root folder
6.Add either the minimal or virtual hosting config at the bottom of Apache httpd.conf file 

Assumptions
a)Your website root directory is called django (This was initially called project but i changed mine to django)
b)Your folder containing wsgi.py file is called project which resides in your website root directory
c)Your static folder resides in your website root directory and it is called static
d)For PHP users, the wsgi.py file behaves like the index.php file that bootstraps your website


Update your wsgi.py file to look like this. Replace the path htdocs\\django with your folder names
    import os,sys
    sys.path.append("C:\\xampp\\htdocs\\django")
    from django.core.wsgi import get_wsgi_application
    os.environ.setdefault("DJANGO_SETTINGS_MODULE", "project.settings")
    application = get_wsgi_application()


---APACHE httpd.conf MINIMAL CONFIGURATION(Redirects not working!!!)---
Add the following code to the bottom of apache httpd.conf

LoadModule wsgi_module modules/mod_wsgi.so
Alias /static/ "C:/xampp/htdocs/django/static/"
WSGIScriptAlias /django "C:/xampp/htdocs/django/project/wsgi.py"
<Directory C:/xampp/htdocs/django>
Order deny,allow
Allow from all
</Directory>

7.Visit your website at localhost/django OR if you are on port 81,use localhost:81/django



---APACHE httpd.conf WSGI VIRTUAL HOST CONFIGURATION(Redirect works,PDF report not working!!!)---

1.First append the following DNS entry to C:\Windows\System32\drivers\etc\hosts file and save
localhost:81      www.djangodev.com

2.Append the following config to apache httpd.conf file.If you have loaded the wsgi_module,dont load it again
LoadModule wsgi_module modules/mod_wsgi.so
<VirtualHost *:81>

  # Site name
  ServerName www.djangodev.com

  # Setup WSGI
  WSGIScriptAlias / "C:/xampp/htdocs/django/project/wsgi.py"
  <Directory C:/xampp/htdocs/django>
    <Files wsgi.py>
      Order deny,allow
      Allow from all
    </Files>
  </Directory>

  # Static files
Alias /static/ "C:/xampp/htdocs/django/static/"
</VirtualHost>

3.In your browser say firefox, setup the IP address as follows
IP = localhost, Port = 81(use your port here)

4.Visit your website at www.djangodev.com

"""
